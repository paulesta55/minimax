# minimax
minimax algorithm for estanocandassamy project

# Get Started

you need estanocandassamy project to build

```sh
$ cd ~
$ git clone <estanocandassamy>
$ git clone <minimax>
$ cd /minimax
$ cp -r ../estanocandassamy/src/shared/state ./src/
$ cp -r ../estanocandassamy/src/shared/engine ./src/
$ cmake .
$ make 
$ ./minimax
```
